CDNify.com for WordPress - View and manage your resources, create new resources within WordPress and choose which resource your website should use.

== Description ==

CDNify Manager is the perfect solution for any WordPress website which needs the use of a CDN. This plugin works with the API of cdnify.com

Sign up to CDNify at [cdnify.com](https://cdnify.com/pricing "Signup instantly to CDNify") and use the code AWESOMECDNIFYMANAGER to get 15% off any monthly plan for 3 months.

* View your resources
* Create new resources
* Choose which resource your website should use
* All CSS files, Javascript files and media assets will use the CDN via your choosen resource
* Easily change resources for migration or testing

== Installation ==

1. Upload the "cdnify-manager" plugin directory to the "/wp-content/plugins/" directory.
1. Activate the plugin through the "Plugins" menu in WordPress.
1. Access the "CDNify" menu under "Tools".
1. You will be asked to enter your API key, which can be found here: <a href="https://cdnify.com/learn/api" target="_blank">https://cdnify.com/learn/api</a>
1. You will then be able to view your existing resources, create new resources and choose the resource to use.
